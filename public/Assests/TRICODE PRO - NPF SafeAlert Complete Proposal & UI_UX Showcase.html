Executive Summary

Proposal Overview

This proposal outlines the development of the NPF SafeAlert Minimum Viable Product (MVP), designed to modernize emergency response and citizen vigilance through cutting-edge mobile technology.

Core Objectives

  • Reduce emergency response time
  • Enhance citizen-police trust
  • Data-driven resource deployment

MVP Scope

  • Mobile App for Citizens (Android/iOS)
  • Admin Dashboard for Officers
  • Real-time Geolocation Services

Security Focus

  • End-to-End Encryption
  • Anonymous Reporting Options
  • Strict Access Control (2FA)
Deliverables

Functional Modules

Module Name Description Key Deliverables
User Authentication Secure sign-up and login for citizens, immigrants, and officers using email, phone number, and biometric verification. JWT Auth, 2FA Integration, Biometric API
Emergency SOS One-touch panic button that sends real-time location and user details to the nearest NPF command center. Real-time Socket.io, Geolocation API
Incident Reporting Structured forms for reporting crimes, accidents, or suspicious activities with media upload capabilities (photos/videos). Media Compression, Cloud Storage, Forms
Admin Dashboard Centralized control panel for NPF officers to view alerts, manage dispatch, and analyze crime data. React Admin Panel, Role-based Access
